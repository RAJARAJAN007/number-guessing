# Number Guessing Game - CLI Version

This is the **Console (CLI) version** of the Number Guessing Game.

## Features
- 3 difficulty levels:
  - Easy (1–10, 5 attempts)
  - Medium (1–50, 7 attempts)
  - Hard (1–100, 10 attempts)
- Handles invalid inputs gracefully (exception handling)
- Tracks number of attempts

## How to Run
1. Open a terminal in this folder (`num_cli/`)
2. Run the game.
